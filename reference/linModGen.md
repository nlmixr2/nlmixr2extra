# Generate a Linearization Model From Previous Fit

Generate a Linearization Model From Previous Fit

## Usage

``` r
linModGen(ui, focei = TRUE, derivFct = FALSE)
```

## Arguments

- ui:

  rx model or fit object.

- focei:

  boolean. If TRUE, use FOCEI linearization with individual and residual
  linearization. Default is TRUE.

- derivFct:

  boolean. If TRUE, use normalization derivative factors. Default is
  FALSE.

## Value

rxUi model
