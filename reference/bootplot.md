# Produce delta objective function for boostrap

Produce delta objective function for boostrap

## Usage

``` r
bootplot(x, ...)

# S3 method for class 'nlmixr2FitCore'
bootplot(x, ...)
```

## Arguments

- x:

  fit object

- ...:

  other parameters

## Value

Fit traceplot or nothing.
