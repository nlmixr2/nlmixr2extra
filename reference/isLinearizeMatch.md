# Check Linearization Match

Check Linearization Match

## Usage

``` r
isLinearizeMatch(linFit, tol = 0.1)
```

## Arguments

- linFit:

  fit of linear model.

- tol:

  relative tolerance for matching. Default is 0.05.

## Value

match list for OFV, omega, eta, and residual variance terms
