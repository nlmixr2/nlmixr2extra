# Plot Original Versus Linear Models iObj and Etas

Plot Original Versus Linear Models iObj and Etas

## Usage

``` r
linearizePlot(lin)
```

## Arguments

- lin:

  linear fitting object

## Value

ggplot object

## Details

If the non-linear fit was not FOCEI, the objective functions are not
comparable, hence can be ignored upon comparison.
